Overview
========

This section provides a quick overview to the COMET DSL and Compiler, as well as the other front-ends to express tensor algebra expressions with host programming languages. 
Some internal details of the COMET compiler, including dialects and optimizations are briefly presented for the curious reader.
The current status of support for different backend architectures is discussed, including a road-map for future releases.
